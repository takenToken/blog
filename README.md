## 操作手册
```
# 由于还未使用go的依赖管理机制，只能手工下载依赖包

go get github.com/go-sql-driver/mysql
go get github.com/gomodule/redigo/redis
go get github.com/pkg/errors

#独立部署
nohup ./beepkg &
```
